# 旋律配和弦网页

这是一个纯静态网页，可以部署到 Netlify、Vercel、Cloudflare Pages 或 GitHub Pages。

## 最简单分享方式

1. 打开 Netlify Drop: https://app.netlify.com/drop
2. 把整个 `melody-harmonizer-public` 文件夹拖进去。
3. 等待上传完成，Netlify 会生成一个公开网址。
4. 把那个网址发给别人即可。

## GitHub Pages

1. 新建一个 GitHub 仓库。
2. 上传本文件夹里的所有文件。
3. 在仓库 Settings -> Pages 中选择部署分支。
4. 等待 GitHub 生成公开网址。

## 注意

这个网页目前不需要服务器和数据库。所有配和弦逻辑都在浏览器本地运行。
